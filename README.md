反转贪吃蛇
============

使用cocos2d-x lua和code ide编写的贪吃蛇小游戏

###初始化
将cocos2d-x 3.1.1 创建的lua项目中，frameworks/cocos2d-x复制到framesworks

本项目可以使用cocos code ide导入

###游戏规则

当贪吃蛇在屏幕左边的时候，按屏幕位置是正常转向的。

当贪吃蛇在屏幕右边的时候，按屏幕位置转向是反转的，例如转左会变成转右。
